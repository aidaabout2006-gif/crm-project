# CRM - Customer Management Project

A simple Node.js CRM to manage customers, groups, and users.

---

##  Tools

- **Node.js** - Backend 
- **Express.js** - Routing 
- **EJS** - dynamic pages
- **JavaScript** - Frontend 
- **HTML / CSS** - Layout and styling
- **JSON** - Project files

---

##  Features

- User login, register, and forgot Password
- Customer management (add, edit, delete)
- Group management
- Dashboard

## Run Project
1. Copy the project to your computer 
2. Open the project folder in `terminal`
3. Install needed packages: `npm install`
4. Start the server: `node server.js`
5. Open your browser and go to: `http://localhost:3000`
